Optimizing CpGs in viral genome to ZAP

A method for vaccine development

By Andrew Cheung

Background/Rationale

One approach to developing effective vaccines is by attenuating, or weakening, a virus. All organisms are made of the same 4 nucleic acids in their DNA: adenosine (A), cytosine (C), guanosine (G), and thymidine (T). Eukaryotic organisms have evolved to suppress the number of cytosines followed by guanines (CpGs) in their genome. Scientists have discovered that the CpG dinucleotide frequency in eukaryotic genomes is statistically smaller than randomly expected. Viruses that infect these organisms have coevolved to minimize their own CpG frequency to mimic the host genome and to avoid detection from the host immune system. It is hypothesized that the zinc-finger antiviral protein (ZAP) plays a major role in the inhibition of viral RNA, by binding to CpGs and recruiting protein machinery to degrade the viral RNA. Successful attenuation of a viruses have been done by optimizing CpGs.1

How this code works!

This code requires a user input DNA sequence, using only the nucleotides "A", "T", "G", and "C". This DNA strand will be converted into a DNA sequence that contains the maximum number of CpGs possible while preserving the amino acid coding. This can be done since one amino acid has many codon pairings (Ex: Leucine can be coded by 6 different codons!). Recall that three nucleotides will be translated into one amino acid. The wobble position of a codon (3rd nucleotide in a codon) can vary and still potentially encode for the same nucleotide. If leucine is currently encoded as "CTT", this can be changed to "CTC" to increasing the chances of introducing a CpG if the next nucleotide is a "G".

The code will then decide which CpG locations would optimally spread the CpGs to be ~13 nucleotides apart. This has been identified as the optimal spacing for ZAP susceptibility and detection by the host immune response. It is possible for some regions to not allow spacings of CpGs to be 13 nucleotides apart due to the specific protein coding in that region. Once the locations are determined, a viable DNA sequence will be produced, measured by GC content. GC content (the number of Cs and Gs over the number of all nucleotides expressed as a percentage) is an important factor for viability (Ex: Human genomes have an average GC content that ranges from 35% to 60% across 100-Kb fragments).
